The source folder within StarTeam project.

For the Root Folder, leave Folder name blank.

For a sub folder provide a the folder path (ommitting the root folder name) e.g. If in Star Team you have the folder structure: 

root_folder
|____foo1
|    |___bar1
|    |___bar2
|
|____foo2

Then:
To specify the bar1 folder enter foo1/bar1
To specify the foo2 folder enter foo2
