The View of the StarTeam project.

For the Default view, leave View name blank.

To specify a child view, then the fully qualified view hierarchy can be entered separated by slashes. e.g. If in Star Team you have the view hierarchy: 

root-view
|____release-1
|    |___sandbox-1
|    |___sandbox-2
|
|____release-2
     |___sandbox-1
     |___sandbox-2

Then:
To specify sandbox-1 in the release-1 branch enter root-view/release-1/sandbox-1
To specify the release-2 branch enter root-view/release-2
